# Micronaut JUnit Jupiter

JUnit 5 extensions & helpers that make working with Micronaut easier.

## Installation

Add the dependency to your build:

```kotlin
// build.gradle.kts
dependencies {
    testImplementation("com.pkware.micronaut:junit-jupiter:1.0.0")
}
```

The extensions are **automatically discovered** via Java's `ServiceLoader` mechanism. No explicit configuration needed.
